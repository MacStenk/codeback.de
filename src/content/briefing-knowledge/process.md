# CodeBack Prozess & Next Steps

## How We Work

### 1. Discovery Phase
**Duration:** 1 Call (15-30 Min)

**We discuss:**
- Your business & goals
- Current website situation
- Budget & timeline
- Which service fits best

**You get:**
- Clear recommendation
- Honest assessment
- Next steps

### 2. Briefing Phase
**Duration:** 1-2 Days

**You provide:**
- Business information
- Content (texts, images)
- Competitor examples
- Design preferences

**We provide:**
- Structured template
- Clear checklist
- Support if needed

### 3. Development Phase
**Duration:** 2-8 weeks (depending on service)

**What happens:**
- Regular updates (every 3-5 days)
- GitHub access to see progress
- Questions via Email/Chat
- Transparent process

**You see:**
- Live preview links
- Code commits
- Progress updates

### 4. Review Phase
**Duration:** 3-7 days

**Included revisions:**
- AI-Discovery: 1 round
- System-Entwicklung: 2 rounds
- Migration: 2 rounds

**We adjust:**
- Content
- Design details
- Functionality
- Nothing major (scope)

### 5. Launch Phase
**Duration:** 1 day

**You get:**
- Full GitHub access
- Deployment setup
- Documentation
- Training (if needed)

**We ensure:**
- Everything works
- You understand it
- Smooth handover
